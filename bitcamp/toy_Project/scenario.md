## 스마트폰 리뷰 프로그램 시나리오

### 1.	생성자 선택  
①	실행 : 입력 받아 놓은 데이터 읽어오기  
②	기존데이터 추가/수정/삭제 가능 
-	데이터 CRUD 작업

### 2.	소비자 선택  
①	생성자 데이터 로드  
②	목록에서 원하는 데이터 선택(읽기)  
③	평점/평균으로 목록 로드  
④	데이터 CRUD 작업 – 리뷰/평점을 작성
 
_범위를 좁게 잡아서 크게_
